Projeto Login

Projeto construído para treinar campos e credenciais e mobile-first

Saiba mais...

O Projeto Login, baseado no Curso em Vídeo do professor Gustavo Guanabara, foi um exercício prático para criar uma tela de login responsiva utilizando apenas HTML5 e CSS3. O projeto abordou a estruturação do formulário de login, incluindo campos para usuário e senha, além da estilização para um visual moderno e funcional, sendo totalmente adaptável a diferentes tamanhos de tela. Auxiliou no desenvolvimento de habilidades fundamentais na criação de interfaces de autenticação, combinando usabilidade e design responsivo, ideal para o aprimoramento do conhecimento em desenvolvimento web front-end.​​
